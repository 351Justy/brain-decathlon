📋 Brain Decathlon とは

Brain Decathlon（ブレイン・デカスロン）は、9種類のパズルに挑戦する毎日の脳トレです。

円周率を暗記しながら、様々なパズルを解いて脳を鍛えましょう！

π（円周率）チャレンジ

用紙上部の罫線エリアに、覚えている円周率の桁数を書き込みましょう。毎日少しずつ覚える桁数を増やしていきましょう！

1️⃣ ビルパズル

左側の数字の組み合わせを使って、右側の数字を作る計算式を考えます。

ルール

  • 左側の数字をすべて1回ずつ使用
  • 四則演算（＋−×÷）を使用可能
  • 括弧を使用可能

2️⃣ 賢賢（KenKen）

数独に似たパズルです。太線で囲まれた領域の数字を計算すると、左上の数字になります。

ルール

  • 各行・各列に1〜6の数字を1つずつ配置
  • 太線で囲まれたケージ内の数字を指定された演算で計算すると、ケージの左上の数字になる

3️⃣ マッチ棒パズル

マッチ棒を動かして、正しい等式を作ります。

ルール

  • 指定された本数のマッチ棒だけを動かす
  • マッチ棒を取り除いたり追加したりはしない

4️⃣ 覆面算

図形に隠された数字を推理します。

ルール

  • 同じ図形には同じ数字が入る
  • 異なる図形には異なる数字が入る
  • 縦・横の計算が成り立つように数字を当てはめる

5️⃣ カウントダウン

与えられた数字と演算子を使って、目標の数を作ります。

ルール

  • 各数字は1回だけ使用可能
  • 四則演算を使用
  • すべての数字を使う必要はない

6️⃣ ミニナンプレ

6×6の数独パズルです。

ルール

  • 各行に1〜6の数字を1つずつ配置
  • 各列に1〜6の数字を1つずつ配置
  • 各2×3ブロックに1〜6の数字を1つずつ配置

7️⃣ 計算パズル

縦横の計算式がすべて成り立つように空欄を埋めます。

ルール

  • 横の計算式が成り立つように数字を入れる
  • 縦の計算式も同時に成り立つ

8️⃣ 合計パズル

行と列の合計が指定された数になるように数字を配置します。

ルール

  • 各行の数字の合計が左端の数字になる
  • 各列の数字の合計が上端の数字になる

9️⃣ 迷路

スタートからゴールまでの道を見つけます。

ルール

  • 壁を通らずに進む
  • 最短経路を見つけなくてもOK
💡 ヒント：行き止まりを見つけて消していく方法が効果的です
